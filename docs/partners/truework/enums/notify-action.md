notification types

```graphql
enum NotifyAction {
  CREATED
  DELETED
  UPDATED
  HIRED
  TERMINATED
}
```

### Values

#### `CREATED`

object was created

#### `DELETED`

object was deleted

#### `UPDATED`

object was updated

#### `HIRED`

for role - role was hired

#### `TERMINATED`

specific to role - role was terminated
