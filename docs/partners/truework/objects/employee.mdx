Employee record, logically matches a person

```graphql
type Employee implements NotifySubject, Node {
  id: ID!
  mongoId: ID!
  ssn: String
  fullName: String
  firstName: String
  lastName: String
  dob: String
  email: String
  phone: String
  homeAddress: Address
  roles: [Role!]
  consent: EmployeeConsent
  employmentHistory: [EmploymentHistoryRecord]
}
```
