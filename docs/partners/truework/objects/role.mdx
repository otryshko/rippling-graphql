Employment position

```graphql
type Role implements NotifySubject, Node {
  id: ID!
  dbId: String
  company: Company
  employee: Employee
  payroll: RolePayroll
  startDate: Date
  endDate: Date
  state: RoleState
  job: Job
}
```
