Client company representation

```graphql
type Company implements NotifySubject, Node {
  id: ID!
  dbId: String
  name: String
  taxInfos: [CompanyTaxInfo!]!
  status: CompanyStatus
  consent: ConsentState
  address: Address
  hasHris: Boolean
  hasPayroll: Boolean
}
```
